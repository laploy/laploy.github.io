ISSUE #1 - ตุลาคม

สวัสดีผท่าผู้เคยเรียนกับเกรทเฟรนด์

บางท่านอาจห่างหายกันไปนาน แต่ก็ยังมีบางท่านที่ยังติดตามผ่านเเพจเกรทเฟรนด์กันอยู่ ซึ่งเพจยังคงอัพเดตทั้งข่าวสารและคอร์สต่างๆ ให้ทันสมัยมาโดยสม่ำเสมอนะคะ ในเวลานี้มีคอร์สที่เพิ่งเปิดใหม่ด้วย นั่นคือ คอร์ส Essential ML.NET และ คอร์ส Essential Angular and TypeScript ค่ะ

คอร์ส Essential ML.NET

สร้างงาน Machine Learning ในแบบฉบับของนักพัฒนา .NET ให้ใช้งานได้โดยไม่จำกัดอุปกรณ์และแพลตฟอร์ม เหมาะสำหรับผู้ที่สนใจ หรือผู้ที่เคยผ่านการอบรมคอร์ส Microsoft Azure Machine Learning Express มาแล้ว

คอร์ส Essential Angular and TypeScript

พัฒนาเว็บด้วย Angular Framework โดยใช้ภาษา TypeScript เชื่อมต่อกับ Service ของระบบอื่นๆ เหมาะสำหรับผู้ที่สนใจ หรือผู้ที่ผ่านการอบรมคอร์ส Advanced ASP.NET Core API and MVC มาแล้ว
